# CPSC 483 Final Project: Digital Distraction vs Academic Performance

THis project will apply every concept that have been learned in CPSC 483 - Machine Learning in California State University, Fullerton

## Project Goal

Using Naive Bayes, Logistic Regression, and Decision Tree to predict the effect of Digital Distraction on a student academic performance level

## How to use the program: 
1) download the dataset with the link provided 
Original dataset can be downloaded at https://www.kaggle.com/datasets/neurocipher/digital-distraction-vs-academic-performance
2) extract the data set in the same file directory as the jupyter notebook file 
3) run the jupyter notebook using python environment 3.12.10
